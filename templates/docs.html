Technical resource for the Influenze Relay automation system. Understand how data flows between Google Sheets and WhatsApp Web.

Influenze Relay is a secure, internal-only automation system. It orchestrates three isolated components without using public APIs, ensuring data sovereignty and session privacy.

Data Source

The system expects a specific schema in the connected Google Sheet. Ensure your sheet has the following headers:

[ "Name", "Niche", "Age", "Category", "Rate", "WhatsApp Number", "Last Contacted" ]

Brand Input

Parameter Description
Brand Name Name of the client brand.
Required Niche Filters creators by this exact niche string.
Budget Max budget available per creator.

Message Logic

We calculate the fair offer price dynamically. This prevents overcommitting budget while ensuring creators get their asked rate if within budget. The total payout is then dynamically multiplied by the number of deliverables requested for the overarching campaign.

Base Price = MIN( Brand Budget, Creator Rate )
Total Payout = Base Price × Deliverables Count

If the product is non-retainable, a specific {{retainable_line}} text is automatically injected into the Jinja2 template specifying that the product must be returned safely upon completion.
